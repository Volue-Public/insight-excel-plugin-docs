Getting started
---------------

This is the help page for the Volue Insight API's excel add-in.

This gives you access to the Volue Insight power market forecast and actuals.
You will be able to use our api to search and discover our data inside Excel.

Haven't used our plugin before? Go to the :ref:`step-by-step <excel-step-by-step>` for a more in-depth description on how to use it.
